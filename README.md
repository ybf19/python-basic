# Python Basic - Aprendiendo Python 🐍

![Python](https://img.shields.io/badge/Python-3.11+-blue.svg)

Este repositorio documenta mi viaje aprendiendo Python desde cero.

## 📚 Estructura de aprendizaje

| Carpeta | Tema |
|---------|------|
| 01_variable_y_tipos | Variables y tipos de datos básicos |
| 02_condicionales | if, elif, else |
| 03_bucles | for, while |
| 04_estructuras_datos | listas, tuplas, diccionarios |
| 05_strings_avanzado | manipulación de strings |
| 06_funciones | creación y uso de funciones |
| 07_errores_excepciones | manejo de errores |
| 08_archivos | lectura/escritura de archivos |
| 09_poo | programación orientada a objetos |
| 10_modulos_paquetes | módulos y paquetes |
| 11_librerias_populares | numpy, pandas, matplotlib |
| proyectos | Proyectos prácticos |

## 🚀 Cómo usar

1. Clona el repositorio
2. Cada carpeta contiene archivos \`.py\` con ejemplos y ejercicios
3. Para instalar dependencias: \`pip install -r requirements.txt\`
